Subir interface de consulta a api em FLASK - porta 5000

1 - Ambiente do flask
    Verificar se os pacotes estão instalados
        pip3 list | grep -i flask
    Instalar
        pip3 install flask
        pip install json2table

2 - Criar arquivo app.py e vero o conteudo do mesmo para exeucção do comando abaixo:
    flask run
